# Elyscents Homepage Clone

A responsive replica of the Elyscents homepage built using React.js and Bootstrap. The project demonstrates component-based architecture, React hooks, routing, protected routes, and dynamic data rendering using a dummy JSON API.

## Tech Stack
- React.js
- JavaScript (ES6)
- Bootstrap
- React Router
- JSON API

## Features
- Responsive homepage layout
- Reusable React components
- Client-side routing using React Router
- Protected routes for restricted pages
- Dynamic content rendering using API data
- Clean and modular component structure
